# 💰 Tax Dashboard — $30K Income · California

A visual, single-page tax breakdown and savings board for a $30,000 income in California. Built as a clean HTML artifact with no dependencies.

## What's Inside

- **Income & Tax Breakdown** — Federal + CA state taxes, deductions, and effective rate at a glance
- **Late Filing Penalty Summary** — How the $600 penalty was calculated and what it means
- **First-Time Abatement (FTA) Guide** — Step-by-step walkthrough to request the IRS penalty waiver
- **Savings & Credits Board** — All available tax-saving options including Federal EITC, Traditional IRA, HSA, Saver's Credit, and more

## Live Preview

> Enable GitHub Pages: `Settings → Pages → Deploy from main branch`
> Then visit: `https://<your-username>.github.io/tax-dashboard`

## Usage

No build step. No dependencies. Just open `index.html` in any browser.

```bash
git clone https://github.com/<your-username>/tax-dashboard.git
cd tax-dashboard
open index.html
```
